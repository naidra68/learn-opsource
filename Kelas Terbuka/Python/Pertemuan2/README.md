# Pertemuan2 - Instalasi (Python Tutorial)

Sebelum memulai belajar Python, kita harus menginstall terlebih dahulu python dan visual studio code di komputer atau laptop kita.

## Install Python

Untuk menginstall python bisa lihat step berikut :

1. Buka web browser (chrome, firefox, brave, edge, etc).
2. Buka [Python Web Page](https://www.python.org/downloads/).
3. Buka hasil Download Python tadi.
4. Add python.exe to PATH (dicentang).
5. Install Now.
6. Tunggu sampai selesai.
7. Disable path length limit di klik.
8. Selesai.

Untuk mengecek apakah python sudah terinstall bisa cek diterminal sebagai berikut :

1. Ketik `terminal` di search atau Ketik `Command Prompt`.
2. Ketik `python --version`
3. Jika ada tulisan versi python maka berhasil diinstal.


## Install VS Code

Untuk menginstall VS Code bisa lihat step berikut :

1. Buka web browser (chrome, firefox, brave, edge, etc).
2. Buka [VS Code Web Page](https://code.visualstudio.com/).
3. Buka hasil Download VS Code tadi.
4. Accept Install.
5. Next.
6. Centang semua Add "Open with Code".
7. Next Install.


Setelah masuk ke VS Code dan membuat folder dan file `Main.py` maka akan disuruh install extention python di VS Code, silahkan install saja. Fungsinya adalah untuk install package python agar bisa digunakan di VS Code.
