## Kenapa Python Populer?

Karena kemudahan Syntax penulisan pyhton lebih simple dari bahasa pemrograman lain. Misalnya, untuk mengetik tulisan <b>Hello World</b> pada bahasa lain memerlukan syntax yang terbilang rumit bagi pemula.

Perhatikan beberapa penulisan teks Hello World pada masing-masing bahasa pemrograman seperti `C++` dan `Java` berikut ini :

```c++
#include <iostream>

int main(int argc, char const *argv[])
{
    std::cout << "Hello World" << std::endl;
    return 0;
}
```

```java
class Main{
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```

Sedangkan untuk menulis <b>Hello World</b> di dalam Python, hanya perlu menggunakan sedikit syntax.

```python
print("Hello World")
```

## Pendiri Python

Guido van Rossum merupakan penemu dan pencipta bahasa pemrograman Python, ditemukan pada tahun 1989. Rossum membuat Python setelah bertemu dengan seseorang bernama Andrew S. Tenenbaum yang bertanya kepada Rossum tentang bahasa yang dapat dimengerti Amoeba (ini berujuk kepada komputer server).

Tenenbaum sedang membangun distributed operating system untuk komputer pada lab nya dan meminta Rossum untuk membuatnya.

Guido van Rossum adalah jagoan pada bahasa pemrograman ABC dan Modula+ yang merupakan bahasa interpreter yang dapat berpindah-pindah platform.
